# Hospital Patient Treatment Analysis (Excel Project)

## Overview
This project focuses on analyzing hospital patient treatment data using Microsoft Excel to extract meaningful insights and improve decision-making.

## Key Features
- Data cleaning and preprocessing
- Use of pivot tables for analysis
- Interactive dashboard creation
- Visualization using charts and graphs

## Tools Used
- Microsoft Excel

## Project Files
- hospital_patient_treatment_dataset.xlsx – Dataset and analysis
- dashboard.pdf – Dashboard pdf
  
## Insights 
- Surgery gives best recovery
- Higher cost → better outcomes
- Middle-aged recover faster
- Recovery varies by doctor & department
